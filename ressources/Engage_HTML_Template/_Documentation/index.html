Template Features

  1. 4 Design Concepts – There are 4 unique designs with 15+ different Demos.
  2. Ultra Responsive & Retina Ready – Engage is 100% responsive & retina ready, each and every element including the sliders are fully responsive.
  3. 3 different menu variants – There are 3 types of menu in this template named overlay, push menu and sidebar menu.
  4. Parallax Sections – Engage is built using Parallax javascript. This enables to make your site more appealing and creative.
  5. Custom Backgrounds – Engage allows you to set a custom background for each section. A super-quick way to get creative with your site.
  6. 5 slider variants – There are 5 beautifully designed sliders in the template.
  7. Cube Portfolio - Included Cube Portfolio, You save $16.00.
  8. Ajax Contact Form - The elements of the contact form is perfectly styled and very easy to customize it.
  9. Browser Compatibility - This template supports all the major browsers including IE9+, Chrome, Safari, Firefox.
  10. Clean, Faster, W3C Validate - The code is well written using the best resources available and also all the files are well commented to make your work much easier and standard validates in W3C school.

One important note before you start working on this HTML theme! Although the theme should work just fine "straight from the desktop", some of the theme features may require a real web server (local or online) to work properly. So keep that in mind!



HTML Structure

It's a one page template so there are many sections and sub sections but all the sections have been divided into 12 "main" sections :

#navigation, #home, #about-us, #parallax-section-1, #team, #services#portfolio, #testimonials, #clients, #pricing #blog and #page_footer.

.container determines the page width and also centralize it. You can adjust this width from css/style.css file.

  • Portfolio

    We are using Cube Portfolio for the portfolio section. It's a jQuery grid plugin that provides powerful portfolio system.

    There are many animation & ajax page styles in the cube portfolio, You can apply as per your requirement. For more info you can read the cube portfolio documentation here.


    Portfolio Filter - All projects are placed in #portfolio and every project is a single <div>. Every project has .cpb-item class which will be used in filtering them. A single project can has multiple categories and they need to be separated with space. There isn't any naming rules but you should use only lowercase and single word (use dash to replace spaces) names to avoid any possible issues.

    Basic template:

    <div id="portfolio-grid-container" class="cbp-l-grid-projects" data-caption="overlayBottomReveal" data-animation="slideLeft" data-gaphorizontal="38" data-gapvertical="50">
    <!--Project 1-->
    <div class="cbp-item graphic identity">
    <a href="ajax/project-1.html" class="cbp-caption cbp-singlePage" data-title="Project Title">
    <div class="cbp-caption-defaultWrap">
    <img src="[your project image url] alt="">
    </div>
    <div class="cbp-caption-activeWrap">
    <div class="cbp-l-caption-alignLeft">
    <div class="cbp-l-caption-body">
    <div class="cbp-l-caption-title">Project Title here</div>
    <div class="cbp-l-caption-desc">Project Decription here</div>
    </div>
    </div>
    </div>
    </a>
    </div>
    <!--/ Project 1-->
    </div>



    Example of filter:

    <!--Filter-->

  • Contact Form

    Setting up the contact form is really simple task. All you need to do is to open php/contact.php file and replace $your_email_address(Line no- 54) value with your own one and that's it!


  • Google Map

    The contact section map can be configure in js/custom.js file, You need replace the Latitude and Longitude Value ( Line No. 808) for your location also you need to change the Latitude and Longitude Value ( Line No. 821) for the google map marker position. In case you need any help, you can read more here.
